Meditation Timer 1.1 Help


What's new:

• You can now set periodic chimes and a delay before the session starts

• You can now export your session to iTunes to burn or put on an iPod

• The interface has been improved

• The internals revamped (ok maybe you won't notice that one).


Adding a delay before starting or periodic chimes:

To gain control over these new settings simply click the triangle next to 'More Options'. The window will enlarge to show settings for a start delay and period chimes.


Exporting a Session to iTunes:

Exporting a session to iTunes is easy with Meditation Timer 1.1. Simply set up the session you'd like to export by setting the intervals and chimes you'd like, and then select 'Export to iTunes...' from the file menu.  Note that it can take several minutes to export.  After Meditation Timer finishes creating your custom audio file, it will automatically import the file and ask you if you'd like to delete the original.  Click yes if you know iTunes is set to automatically copy files to your library. You can now use iTunes to burn the session or load it onto an iPod.


Adding Sound Files:

You can add your own sound files easily by adding an MP3 to the resources folder. Right click meditation timer, select "Show Package Contents", and open the 'Contents/Resources' folder. Drop MP3s here and Meditation Timer will automatically detect them
